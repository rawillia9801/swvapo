# AGENTS.md

## Project standard
This portal must look premium, cohesive, and production-ready. Avoid generic AI-looking dashboard design.

## Visual language
- Warm premium palette: cream, latte, camel, bronze, cocoa
- Refined cards and surfaces
- Tasteful glass effects only where they improve the design
- Strong spacing consistency
- Polished typography hierarchy
- Avoid black-heavy UI
- Avoid plain generic all-white dashboard styling

## Layout rules
- Use a consistent left sidebar shell
- Do not introduce a portal top nav unless explicitly requested
- Keep all portal pages visually unified
- Maintain responsive behavior for desktop, tablet, and mobile

## UX rules
- Every clickable control must either work or be clearly disabled with explanation
- Include loading, empty, and error states
- Forms must be readable and polished
- Tables must be clear, aligned, and professional
- Avoid clutter

## Code output rules
- Return full complete replacement files for changed files
- Never return patch fragments or splice instructions
- Keep code organized and readable
- Prefer reusable components where appropriate
- Do not invent database fields or fake backend wiring

## Finish pass
Before finalizing:
- standardize spacing
- standardize typography
- standardize buttons and badges
- verify sidebar consistency
- verify mobile layout
- remove anything that looks unfinished or placeholder
